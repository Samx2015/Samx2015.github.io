Pembantu layar menu bar Mac

MacHelper Pro

Aplikasi bilah menu yang ringan untuk kontrol layar MacBook, lampu latar keyboard, Auto Wi-Fi Switch, menu konteks Finder, Safe lokal, dan Secure Pocket.

Blackout Menjaga layar bawaan MacBook tetap gelap saat mirroring ke monitor eksternal.
Kontrol Mirror, redup, tidur, kecerahan, kontras, dan pintasan dari menu bar.
Finder + Safe Buat file dan folder, tempel item, buka path di app, serta enkripsi atau dekripsi Safe lokal.
Tanpa jendela Dock Tanpa kata sandi Wi-Fi Safe lokal Uji coba 14 hari
Apa ini?

Semua yang dilakukannya.

  • Auto BlackoutMenjaga layar bawaan MacBook tetap gelap saat mencerminkan layar eksternal.
  • Kontrol layarGanti pencerminan, redupkan layar sampai tombol ditekan, tidurkan Mac, dan sesuaikan kecerahan atau kontras dari bilah menu.
  • Lampu latar keyboardSesuaikan kecerahan keyboard bawaan MacBook dengan slider.
  • Auto Wi-Fi SwitchSaat aktif, memantau jaringan Wi-Fi tersimpan dan beralih ke access point yang dikenal lebih baik saat sinyal saat ini turun.
  • Menu konteks FinderBuat file dan folder, tempel item yang disalin, salin path, atau buka pilihan Finder di TextEdit, Terminal, Codex, Claude, VSCode, atau app kompatibel lain.
  • SafeEnkripsi file atau folder terpilih menjadi satu file .safe lokal, tambahkan ke Safe yang ada, atau dekripsi Safe kembali ke folder.
  • Secure PocketSimpan catatan pribadi kecil, cuplikan clipboard, dan file di brankas terenkripsi lokal, dengan ekspor dan impor paket .mhpocket.
  • Dasar privasiTanpa akun, tanpa analitik, tanpa kata sandi Wi-Fi, dan tanpa jendela Dock.
Apa keunggulannya?

Lebih sedikit repot di meja.

  • Hanya gelap saat perlu gelap saat mirroring, normal saat layar MacBook menjadi bagian ruang kerja
  • Aksi harian cepat ubah mirroring, redup sampai tombol ditekan, tidur, atur level layar, dan biarkan Wi-Fi berganti saat perlu
  • Berguna, tidak berisik berjalan dari menu bar tanpa jendela Dock, akun, atau analytics
Untuk meja kerja dibuat untuk MacBook yang sering tersambung ke layar eksternal
Untuk kontrol cepat menu adalah aplikasinya; tidak ada jendela ekstra
Untuk privasi tanpa akun, analytics, atau kata sandi Wi-Fi; feedback terbuka sebagai draft email yang bisa ditinjau
Cara mendapatkan

Unduh dan coba 14 hari.

  • Unduh satu aplikasi untuk Apple Silicon dan Intel Mac
  • Instal seret MacHelper Pro ke folder Applications
  • Coba semua fitur Pro aktif selama uji coba
macOS 13.0 atau lebih baru arm64 + x86_64
  1. 1 Unduh DMG.
  2. 2 Seret app ke folder Applications.
  3. 3 Jalankan dari folder Applications.
Ditandatangani developer dan dinotariskan Apple.
