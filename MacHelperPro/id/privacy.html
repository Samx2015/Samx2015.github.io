Kebijakan Privasi MacHelper Pro

Tanggal berlaku: 2 Juni 2026

MacHelper Pro adalah utilitas bilah menu macOS yang mengutamakan data lokal untuk mengontrol layar bawaan dalam pengaturan Mac yang dicerminkan dan secara opsional membantu Mac beralih ke Wi‑Fi tersimpan yang lebih kuat.

Versi terjemahan disediakan untuk kemudahan. Versi bahasa Inggris adalah versi yang berwenang dari kebijakan ini.

Tidak ada akun aplikasi Tidak ada MacHelper Pro proses masuk atau profil pengguna.
Tidak ada analitik Tidak ada pelacakan peristiwa, pelacakan sesi, piksel iklan, atau telemetri penggunaan.
Tidak ada kata sandi Wi-Fi Wi-Fi peralihan menggunakan macOS detail jaringan; MacHelper Pro tidak membaca kata sandi.

Pengumpulan Data

MacHelper Pro tidak mengumpulkan, menyimpan, menjual, membagikan, atau mengirimkan informasi pribadi melalui fitur manajemen tampilannya. Xintech LLC tidak menerima informasi tentang tampilan, pengaturan, penggunaan, atau perangkat Anda dari aplikasi kecuali Anda sengaja mengirimkannya melalui dukungan email atau mengaktifkan lisensi.

  • Tidak diperlukan akun MacHelper Pro.
  • Tidak ada analisis, SDK pelacakan, SDK periklanan, atau SDK pelaporan kerusakan yang disertakan.
  • Tidak ada telemetri latar belakang yang dikirim ke Xintech LLC.

Data Lokal

MacHelper Pro mungkin menyimpan sejumlah kecil data preferensi lokal dan log diagnostik lokal terbatas di Mac Anda. Ini hanya digunakan untuk membuat aplikasi berperilaku konsisten di antara peluncuran dan untuk menyiapkan draf laporan bug saat Anda memilih untuk melaporkan bug.

Preferensi lokal mungkin termasuk:

  • Apakah pemadaman otomatis diaktifkan atau dinonaktifkan.
  • Apakah pencerminan internal harus diaktifkan saat peluncuran.
  • Pintasan keyboard khusus untuk tindakan menu MacHelper Pro.
  • Apakah Auto Wi-Fi Switch diaktifkan dan ambang batas kekuatan sinyalnya.
  • Pengidentifikasi tampilan lokal yang digunakan untuk mengingat tampilan eksternal pilihan untuk pencerminan.
  • Log kecil berisi pesan diagnostik MacHelper Pro yang digunakan untuk draf laporan bug.
  • Penanda mulai uji coba dan berakhirnya uji coba disimpan secara lokal dan di Keychain untuk mengelola akses uji coba.
  • Kunci lisensi dan pengidentifikasi instans lisensi Lemon Squeezy disimpan di Keychain Mac Anda jika Anda mengaktifkan lisensi.
  • Stempel waktu lisensi non-rahasia, seperti waktu validasi terakhir yang berhasil, digunakan untuk perilaku tenggang offline.

Aplikasi ini menggunakan macOS API tampilan untuk memeriksa tata letak tampilan, mendeteksi pencerminan, menyesuaikan kecerahan layar bawaan, meredupkan sementara tampilan aktif saat Anda memilih "Dim Displays Until Key Press," kelola Start at Login saat Anda mengaktifkannya, dan memulihkan nilai kecerahan aman tetap bila diperlukan. Untuk peredupan sementara, MacHelper Pro memeriksa apakah tombol ditekan saja sehingga dapat mengembalikan tampilan; itu tidak mencatat nama kunci, teks yang diketik, atau riwayat keyboard. MacHelper Pro tidak menyimpan kecerahan Anda sebelumnya.

Status Wi-Fi dan Izin Lokasi

Auto Wi-Fi Switch bersifat opsional. Jika diaktifkan, MacHelper Pro menggunakan API macOS CoreWLAN publik untuk membaca nama Wi-Fi saat ini, alamat titik akses, kekuatan sinyal, dan pita, serta untuk memindai jaringan gabung otomatis tersimpan yang mungkin merupakan alternatif yang lebih baik.

macOS mungkin memerlukan izin Lokasi sebelum aplikasi dapat melihat Wi-Fi nama jaringan dan alamat titik akses. MacHelper Pro menggunakan izin itu hanya untuk detail Wi-Fi yang digunakan oleh menu dan Auto Wi-Fi Switch. Aplikasi tidak mengumpulkan riwayat lokasi, tidak mengirimkan detail Wi-Fi ke Xintech LLC, dan tidak membaca, menyimpan, atau mengirimkan kata sandi Wi-Fi.

Aktivitas Jaringan

Kontrol tampilan normal MacHelper Pro tidak memerlukan koneksi jaringan. Aplikasi dapat menghubungi umpan pembaruan Sparkle publik ketika pemeriksaan pembaruan otomatis diaktifkan atau ketika Anda memilih "Check for Updates..." dari menu.

Pemeriksaan pembaruan hanya digunakan untuk mengetahui apakah versi yang lebih baru tersedia. Mereka tidak digunakan untuk periklanan, analitik, pelacakan akun, atau pelacakan penggunaan.

Jika Anda memilih "Activate License...", MacHelper Pro menghubungi Lemon Squeezy untuk mengaktifkan, memvalidasi, atau menonaktifkan lisensi Anda. Permintaan lisensi mengirimkan kunci lisensi dan nama instans Anda berdasarkan nama komputer lokal Mac Anda, seperti "MacHelper Pro di Sam's Mac", sehingga Anda dapat mengenali dan menonaktifkan Mac tersebut nanti.

Lemon Squeezy dapat mengembalikan lisensi, produk, pesanan, dan metadata pelanggan yang diperlukan untuk memverifikasi lisensi. MacHelper Pro hanya menyimpan kunci lisensi, pengidentifikasi instance lisensi lokal, penanda uji coba lokal, dan stempel waktu validasi lokal. Permintaan lisensi tidak digunakan untuk analisis, periklanan, atau pelacakan penggunaan.

Laporan Bug, Permintaan Fitur, dan Masukan

Item menu "Report a Bug...", "Suggest a Feature...", dan "Send Feedback..." membuka draf di aplikasi email default Anda yang ditujukan ke Xintech LLC. Aplikasi tidak mengirim email secara otomatis. Anda dapat meninjau, mengedit, atau menghapus draf sebelum mengirimkannya.

Draf tersebut mungkin berisi ringkasan diagnostik singkat sehingga dukungan dapat memahami konteksnya: versi aplikasi, versi macOS, arsitektur prosesor, status Auto Blackout, status pencerminan bawaan, dan status aplikasi saat ini. Alamat email Anda hanya dapat dilihat oleh Xintech LLC jika Anda mengirim email tersebut.

Draf laporan bug juga menyertakan kutipan terbatas dari log diagnostik MacHelper Pro terbaru. Log ini mungkin mencakup status tampilan lokal, detail status aplikasi, nama jaringan Wi-Fi, alamat titik akses, kekuatan sinyal, dan keputusan Auto Wi-Fi Switch. Permintaan fitur dan draf masukan umum tidak menyertakan log.

Diagnostik dan Log

MacHelper Pro menulis pesan diagnostik ke sistem pencatatan log terpadu lokal Apple dan menyimpan log diagnostik lokal aplikasi kecil yang dibatasi. Log ini disimpan secara lokal di Mac Anda. Memilih "Report a Bug..." mengumpulkan kutipan terbatas MacHelper Pro saja dari log lokal aplikasi dan memasukkannya ke dalam draf email.

Kutipan log menggunakan 20 menit terakhir dari MacHelper Pro log jika tersedia. Jika entri terbaru terlalu sedikit, aplikasi dapat melihat kembali hingga 24 jam dan menyertakan entri MacHelper Pro terbaru, dibatasi hingga 180 baris dan 16.000 karakter. Tidak ada yang dikirim kecuali Anda memilih Kirim, dan Anda dapat menghapus kutipan log terlebih dahulu.

Pilihan Anda

  • Anda dapat membiarkan draf masukan tidak terkirim, mengeditnya, atau menghapus detail diagnostik.
  • Anda dapat menghapus ringkasan diagnostik atau kutipan log sebelum mengirim laporan bug.
  • Anda dapat menonaktifkan pemeriksaan pembaruan otomatis di preferensi pembaruan Sparkle.
  • Anda dapat mengaktifkan atau menonaktifkan Auto Wi-Fi Switch dari menu MacHelper Pro.
  • Anda dapat memberikan, menolak, atau mencabut izin Lokasi untuk detail Wi-Fi di macOS Pengaturan Sistem.
  • Anda dapat menggunakan kontrol tampilan Pro selama uji coba tanpa mengaktifkan lisensi.
  • Anda dapat menonaktifkan Mac ini dari jendela lisensi.
  • Anda dapat menghapus preferensi aplikasi lokal menggunakan alat manajemen preferensi macOS biasa.
