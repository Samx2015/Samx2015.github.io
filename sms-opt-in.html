Flexible Timers SMS Opt-In Evidence

This page documents the Flexible Timers SMS opt-in flow for Twilio toll-free verification. Flexible Timers SMS is limited to account phone verification codes and user-created timer/reminder SMS sent only to a signed-in user's own verified, opted-in account phone number. Flexible Timers does not let users message arbitrary third parties, does not send marketing text messages, and does not provide two-way SMS chat.

Opt-In Flow

  1. User signs in to Flexible Timers.
  2. User opens Account Info.
  3. User opens Set Up SMS.
  4. User enters their own phone number.
  5. User checks the SMS consent checkbox.
  6. User can open the SMS Terms and Privacy Policy links from the same screen.
  7. User clicks Continue to request a one-time verification code.
  8. After the phone is verified and opted in, the user may schedule reminder SMS only to that same verified, opted-in account phone number.

Consent Language

I agree to receive SMS verification codes and Flexible Timers reminder messages I schedule for myself at this phone number. Standard message and data rates may apply. Reply STOP to opt out and HELP for help. Message frequency varies. Flexible Timers will not send marketing texts. Consent is not a condition of purchase.

Sample Production Messages

Flexible Timers verification code: 123456. This code expires in 10 minutes. Reply STOP to opt out, HELP for help.

Flexible Timers reminder: You scheduled this SMS reminder for 4:00 PM: Pick up child for after-school piano class. Reply STOP to opt out, HELP for help.

Keyword Responses

Inbound keywords are used only for compliance and account-phone messaging controls. Flexible Timers does not accept general SMS conversations.

STOP response: You are opted out of Flexible Timers SMS. No more messages will be sent. Reply START to opt in again.

HELP response: Flexible Timers sends account verification codes and reminder SMS you schedule for yourself. Help: https://xintechllc.com/FlexibleTimers/support.html. Reply STOP to opt out.

START response: You have opted back in to Flexible Timers SMS messages. Message frequency varies. Reply STOP to opt out, HELP for help.
