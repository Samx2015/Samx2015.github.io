นโยบายความเป็นส่วนตัวของ MacHelper Pro

วันที่มีผล: 13 มิถุนายน 2026

MacHelper Pro เป็นยูทิลิตีแถบเมนู macOS ที่ทำงานภายในเครื่องเป็นหลัก สำหรับควบคุมจอภาพในตัวในการตั้งค่า Mac แบบมิเรอร์ และอาจช่วยให้ Mac สลับไปยัง Wi‑Fi ที่บันทึกไว้ซึ่งมีสัญญาณแรงกว่าได้

มีฉบับแปลเพื่อความสะดวก ฉบับภาษาอังกฤษเป็นฉบับที่มีผลบังคับของนโยบายนี้

ไม่มีบัญชีแอป ไม่มีการลงชื่อเข้าใช้ MacHelper Pro หรือโปรไฟล์ผู้ใช้
ไม่มีการวิเคราะห์ ไม่มีการติดตามกิจกรรม การติดตามเซสชัน พิกเซลโฆษณา หรือการวัดและส่งข้อมูลทางไกลการใช้งาน
ไม่มีรหัสผ่าน Wi-Fi Wi-Fi การสลับใช้ macOS รายละเอียดเครือข่าย MacHelper Pro ไม่อ่านรหัสผ่าน

การรวบรวมข้อมูล

MacHelper Pro จะไม่รวบรวม จัดเก็บ ขาย แบ่งปัน หรือส่งข้อมูลส่วนบุคคลผ่านคุณลักษณะการจัดการการแสดงผล Xintech LLC ไม่ได้รับข้อมูลเกี่ยวกับจอแสดงผล การตั้งค่า การใช้งาน หรืออุปกรณ์ของคุณจากแอป เว้นแต่คุณจะตั้งใจส่งข้อมูลดังกล่าวผ่านการสนับสนุนทางอีเมลหรือเปิดใช้งานใบอนุญาต

  • ไม่จำเป็นต้องมีบัญชี MacHelper Pro
  • ไม่มีการวิเคราะห์ SDK การติดตาม SDK โฆษณา หรือ SDK การรายงานข้อขัดข้องรวมอยู่ด้วย
  • ไม่มีการส่งการวัดและส่งข้อมูลทางไกลในเบื้องหลังไปที่ Xintech LLC

ข้อมูลท้องถิ่น

MacHelper Pro อาจจัดเก็บข้อมูลการตั้งค่าท้องถิ่นจำนวนเล็กน้อยและบันทึกการวินิจฉัยภายในแบบมีขอบเขตบน Mac ของคุณ ใช้เพื่อทำให้แอปทำงานอย่างสม่ำเสมอระหว่างการเปิดตัว และเพื่อเตรียมร่างรายงานข้อบกพร่องเมื่อคุณเลือกที่จะรายงานข้อบกพร่อง

การตั้งค่าท้องถิ่นอาจรวมถึง:

  • ไม่ว่าจะเปิดหรือปิดไฟดับอัตโนมัติ
  • ควรเปิดใช้งานการมิเรอร์ในตัวเมื่อเปิดใช้งานหรือไม่
  • แป้นพิมพ์ลัดที่กำหนดเองสำหรับการทำงานของเมนู MacHelper Pro
  • เปิดใช้งาน Auto Wi-Fi Switch และเกณฑ์ความแรงของสัญญาณหรือไม่
  • ตัวระบุการแสดงผลในตัวเครื่องใช้เพื่อจดจำจอแสดงผลภายนอกที่ต้องการสำหรับการมิเรอร์
  • บันทึกขอบเขตเล็กๆ ของข้อความวินิจฉัย MacHelper Pro ที่ใช้สำหรับร่างรายงานข้อบกพร่อง
  • เครื่องหมายเริ่มต้นการทดลองใช้และหมดอายุการทดลองใช้จัดเก็บไว้ในเครื่องและใน Keychain เพื่อจัดการการเข้าถึงการทดลองใช้
  • รหัสใบอนุญาตและตัวระบุอินสแตนซ์ใบอนุญาต Lemon Squeezy เก็บไว้ใน Keychain ของ Mac หากคุณเปิดใช้งานใบอนุญาต
  • การประทับเวลาใบอนุญาตที่ไม่เป็นความลับ เช่น เวลาตรวจสอบที่สำเร็จครั้งล่าสุด ซึ่งใช้สำหรับการทำงานแบบผ่อนผันแบบออฟไลน์

แอปใช้ macOS API การแสดงผลเพื่อตรวจสอบเค้าโครงจอแสดงผล ตรวจจับการมิเรอร์ ปรับความสว่างของจอแสดงผลในตัว ลดแสงการแสดงผลที่ใช้งานอยู่ชั่วคราวเมื่อคุณเลือก "Dim Displays Until Key Press," จัดการ Start at Login เมื่อคุณเปิดใช้งาน และคืนค่าความสว่างที่ปลอดภัยคงที่เมื่อจำเป็น สำหรับการหรี่แสงชั่วคราว MacHelper Pro ตรวจสอบว่ามีการกดปุ่มเท่านั้นเพื่อให้สามารถคืนค่าจอแสดงผลได้หรือไม่ โดยจะไม่บันทึกชื่อคีย์ ข้อความที่พิมพ์ หรือประวัติแป้นพิมพ์ MacHelper Pro ไม่เก็บความสว่างก่อนหน้าของคุณ

เมนูบริบท Finder และ Safes

การทำงานของ Finder จะทำงานหลังจากคุณเลือกเท่านั้น MacHelper Pro สามารถสร้างไฟล์และโฟลเดอร์ วาง คัดลอกพาธ เปิดรายการที่เลือกในแอปในเครื่อง เข้ารหัสรายการที่เลือกเป็นไฟล์ .safe เพิ่มไปยัง Safe ที่มีอยู่ หรือถอดรหัส Safe ไปยังโฟลเดอร์ที่คุณเลือก

การเข้ารหัสและถอดรหัส Safe อ่านเฉพาะไฟล์ในเครื่องที่เลือกหรือไฟล์ .safe สำหรับการทำงานนั้น รหัสผ่านจะไม่ถูกจัดเก็บ บันทึก อัปโหลด หรือส่งไปยัง Xintech LLC

Secure Pocket

Secure Pocket เก็บโน้ตส่วนตัวสั้น ๆ ข้อความจากคลิปบอร์ด และไฟล์ไว้ในคลังเข้ารหัสในเครื่องบน Mac ของคุณ แพ็กเกจ .mhpocket คือไฟล์ส่งออก/นำเข้าที่เข้ารหัสซึ่งคุณสร้างเอง

Secure Link และ Secure Drop อาจรู้จักลิงก์หรือเตรียมขั้นตอนส่งต่อแบบเข้ารหัสในอนาคต แอปจะไม่อัปโหลดไฟล์ที่เลือกหรือส่งเนื้อหา Secure Pocket เว้นแต่ขั้นตอนในอนาคตจะขอให้คุณเริ่มการถ่ายโอนนั้นอย่างชัดเจน

สถานะ Wi-Fi และสิทธิ์ตำแหน่งที่ตั้ง

Auto Wi-Fi Switch เป็นทางเลือก เมื่อเปิดใช้งาน MacHelper Pro จะใช้ macOS CoreWLAN API สาธารณะเพื่ออ่านชื่อ Wi-Fi ปัจจุบัน ที่อยู่จุดเข้าใช้งาน ความแรงของสัญญาณ และแบนด์ และเพื่อสแกนหาเครือข่ายเข้าร่วมอัตโนมัติที่บันทึกไว้ซึ่งอาจเป็นทางเลือกที่ดีกว่า

macOS อาจต้องได้รับอนุญาตจากตำแหน่งก่อนที่แอปจะสามารถเห็นชื่อเครือข่าย Wi-Fi และที่อยู่จุดเข้าใช้งาน MacHelper Pro ใช้การอนุญาตนั้นสำหรับรายละเอียด Wi-Fi ที่ใช้โดยเมนูและ Auto Wi-Fi Switch เท่านั้น แอปไม่รวบรวมประวัติตำแหน่ง ไม่ส่งรายละเอียด Wi-Fi ไปยัง Xintech LLC และไม่อ่าน จัดเก็บ หรือส่งรหัสผ่าน Wi-Fi

กิจกรรมเครือข่าย

ส่วนควบคุมการแสดงผลปกติของ MacHelper Pro ไม่จำเป็นต้องเชื่อมต่อเครือข่าย แอปอาจติดต่อฟีดอัปเดตสาธารณะ Sparkle เมื่อเปิดใช้งานการตรวจสอบการอัปเดตอัตโนมัติหรือเมื่อคุณเลือก "Check for Updates..." จากเมนู

การตรวจสอบการอัปเดตจะใช้เพื่อเรียนรู้ว่ามีเวอร์ชันใหม่กว่าหรือไม่ ไม่ได้ใช้สำหรับการโฆษณา การวิเคราะห์ การติดตามบัญชี หรือการติดตามการใช้งาน

หากคุณเลือก "Activate License..." MacHelper Pro จะติดต่อ Lemon Squeezy เพื่อเปิดใช้งาน ตรวจสอบ หรือปิดใช้งานใบอนุญาตของคุณ คำขอใบอนุญาตจะส่งรหัสใบอนุญาตของคุณและชื่ออินสแตนซ์ตามชื่อคอมพิวเตอร์ในเครื่อง Mac ของคุณ เช่น "MacHelper Pro บน Sam's Mac" เพื่อให้คุณสามารถจดจำและปิดใช้งาน Mac เครื่องนั้นได้ในภายหลัง

Lemon Squeezy อาจส่งคืนใบอนุญาต ผลิตภัณฑ์ คำสั่งซื้อ และข้อมูลเมตาของลูกค้าที่จำเป็นในการตรวจสอบใบอนุญาต MacHelper Pro จัดเก็บเฉพาะรหัสใบอนุญาต ตัวระบุอินสแตนซ์ใบอนุญาตในเครื่อง เครื่องหมายทดลองใช้ในเครื่อง และการประทับเวลาการตรวจสอบความถูกต้องในเครื่อง คำขอใบอนุญาตไม่ได้ใช้สำหรับการวิเคราะห์ การโฆษณา หรือการติดตามการใช้งาน

รายงานข้อผิดพลาด คำขอคุณสมบัติ และคำติชม

รายการเมนู "Report a Bug...", "Suggest a Feature..." และ "Send Feedback..." จะเปิดฉบับร่างในแอปอีเมลเริ่มต้นของคุณที่ส่งถึง Xintech LLC แอปไม่ส่งอีเมลโดยอัตโนมัติ คุณสามารถตรวจสอบ แก้ไข หรือลบฉบับร่างก่อนส่งได้

ฉบับร่างอาจมีสรุปการวินิจฉัยสั้นๆ เพื่อให้ฝ่ายสนับสนุนเข้าใจบริบท: เวอร์ชันของแอป, เวอร์ชัน macOS, สถาปัตยกรรมตัวประมวลผล, สถานะ Auto Blackout, สถานะการมิเรอร์ในตัว และสถานะแอปปัจจุบัน ที่อยู่อีเมลของคุณจะปรากฏแก่ Xintech LLC เฉพาะเมื่อคุณส่งอีเมลเท่านั้น

ร่างรายงานข้อบกพร่องยังรวมข้อความที่ตัดตอนมาจากบันทึกการวินิจฉัย MacHelper Pro ล่าสุดด้วย บันทึกเหล่านี้อาจรวมถึงสถานะการแสดงผลในเครื่อง รายละเอียดสถานะแอป ชื่อเครือข่าย Wi-Fi ที่อยู่จุดเข้าใช้งาน ความแรงของสัญญาณ และการตัดสินใจ Auto Wi-Fi Switch คำขอคุณลักษณะและแบบร่างข้อเสนอแนะทั่วไปไม่รวมบันทึก

การวินิจฉัยและบันทึก

MacHelper Pro เขียนข้อความการวินิจฉัยไปยังระบบการบันทึกแบบรวมในเครื่องของ Apple และเก็บบันทึกการวินิจฉัยภายในแอปที่มีขอบเขตขนาดเล็ก บันทึกเหล่านี้จะถูกจัดเก็บไว้ในเครื่อง Mac ของคุณ การเลือก "Report a Bug..." จะรวบรวม MacHelper Pro-ข้อความที่ตัดตอนมาจากบันทึกภายในแอปที่มีขอบเขตเท่านั้น และแทรกลงในร่างอีเมล

ข้อความที่ตัดตอนมาจากบันทึกใช้บันทึก MacHelper Pro ช่วง 20 นาทีสุดท้ายเมื่อพร้อมใช้งาน หากมีรายการล่าสุดน้อยเกินไป แอปอาจดูย้อนหลังได้ถึง 24 ชั่วโมงและรวมรายการ MacHelper Pro ล่าสุด โดยมีความยาวสูงสุด 180 บรรทัดและ 16,000 อักขระ ไม่มีการส่งใดๆ เว้นแต่คุณจะเลือกส่ง และคุณสามารถลบข้อความที่ตัดตอนมาจากบันทึกก่อนได้
