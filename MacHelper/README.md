# MacHelper Website

Source for the MacHelper App Store website.

MacHelper shares the same product family and visual system as MacHelper Pro, but
this site is intentionally App Store-oriented:

- no Sparkle appcast or direct-download DMG
- no trial or license purchase copy
- no Auto Wi-Fi switching copy
- Finder context menus are described as the App Store-compatible Finder
  extension feature
- local `.safe` encryption/decryption is described as an App Store-compatible
  local Finder action
