MacHelper Pro 隐私政策

生效日期：2026 年 6 月 13 日

MacHelper Pro 是一款本地优先的 macOS 菜单栏工具，用于在 Mac 镜像设置中控制内建显示屏， 并可选择帮助 Mac 切换到信号更强的已保存 Wi-Fi。

本译文仅为方便阅读而提供。英文版本是本政策的权威版本。

无需 App 账号 无需 MacHelper Pro 登录或用户资料。
无分析追踪 不进行事件跟踪、会话跟踪、广告像素或使用遥测。
不读取 Wi-Fi 密码 Wi-Fi 切换使用 macOS 网络详情；MacHelper Pro 不读取密码。

数据收集

MacHelper Pro 不会通过显示管理功能收集、存储、出售、分享或传输个人信息。除非你主动通过 支持邮件发送信息或激活许可证，否则 Xintech LLC 不会从 App 收到关于你的显示器、设置、使用情况或设备的信息。

  • 无需 MacHelper Pro 账号。
  • 不包含分析、跟踪、广告或崩溃报告 SDK。
  • 不会向 Xintech LLC 发送后台遥测。

本地数据

MacHelper Pro 可能会在你的 Mac 上保存少量本地偏好设置和有限的本地诊断日志。这些数据仅用于让 App 在多次启动之间保持一致行为，并在你选择报告问题时准备错误报告草稿。

本地偏好设置可能包括：

  • 自动黑屏是否开启。
  • 启动时是否启用内建屏幕镜像。
  • MacHelper Pro 菜单操作的自定义键盘快捷键。
  • Auto Wi-Fi Switch 是否开启及其信号强度阈值。
  • 用于记住镜像首选外接显示器的本地显示器标识符。
  • 用于错误报告草稿的一小段 MacHelper Pro 诊断消息日志。
  • 用于管理试用权限的试用开始和试用到期标记，保存在本地和钥匙串中。
  • 如果你激活许可证，许可证密钥和 Lemon Squeezy 许可证实例标识符会保存在 Mac 钥匙串中。
  • 非秘密的许可证时间戳，例如上次成功验证时间，用于离线宽限行为。

App 使用 macOS 显示 API 检查显示器布局、检测镜像、调整内建显示屏亮度、在你选择“调暗显示器直到按键”时临时调暗活动显示器、 在你启用时管理登录时启动，并在需要时恢复固定的安全亮度值。对于临时调暗，MacHelper Pro 只检查是否有按键按下，以便恢复显示器； 它不会记录按键名称、输入的文本或键盘历史。MacHelper Pro 不会保存你之前的亮度。

Finder 上下文菜单和 Safes

Finder 操作只会在你主动选择后运行。MacHelper Pro 可以新建文件和文件夹、粘贴、复制路径、用本地应用打开所选项目、将所选项目加密为 .safe 文件、添加到现有 Safe，或将 Safe 解密到你选择的文件夹。

Safe 加密和解密只会读取该操作所选的本地文件或 .safe 归档。密码绝不会被保存、记录、上传或发送给 Xintech LLC。

Secure Pocket

Secure Pocket 会把小型私人笔记、剪贴板片段和文件存放在 Mac 上的本地加密保险库中。.mhpocket 包是你手动创建的加密导出/导入文件。

Secure Link 和 Secure Drop 可能识别链接或准备未来的加密交接流程。除非未来流程明确要求你开始传输，否则 App 不会上传所选文件或传输 Secure Pocket 内容。

Wi-Fi 状态和位置权限

Auto Wi-Fi Switch 是可选功能。启用后，MacHelper Pro 使用公开的 macOS CoreWLAN API 读取当前 Wi-Fi 名称、接入点地址、信号强度和频段，并扫描可能作为更好替代的已保存自动加入网络。

macOS 可能要求位置权限，App 才能查看 Wi-Fi 网络名称和接入点地址。MacHelper Pro 仅将该权限用于菜单和 Auto Wi-Fi Switch 所需的 Wi-Fi 详情。App 不收集位置历史，不向 Xintech LLC 发送 Wi-Fi 详情，也不读取、存储或传输 Wi-Fi 密码。

网络活动

MacHelper Pro 的常规显示控制不需要网络连接。当自动更新检查开启，或你从菜单选择“检查更新...”时， App 可能会访问公开的 Sparkle 更新 feed。

更新检查仅用于了解是否有新版本。它不会用于广告、分析、账号跟踪或使用情况跟踪。

如果你选择“激活许可证...”，MacHelper Pro 会联系 Lemon Squeezy 来激活、验证或停用许可证。 许可证请求会发送你的许可证密钥，以及基于 Mac 本地电脑名称的实例名称，例如“MacHelper Pro on Sam's Mac”，以便你以后识别并停用该 Mac。

Lemon Squeezy 可能返回验证许可证所需的许可证、产品、订单和客户元数据。MacHelper Pro 只存储许可证密钥、本地许可证实例标识符、 本地试用标记和本地验证时间戳。许可证请求不会用于分析、广告或使用情况跟踪。

错误报告、功能建议和反馈

“报告问题...”、“建议功能...”和“发送反馈...”菜单项会在你的默认邮件 App 中打开一封发给 Xintech LLC 的草稿。 App 不会自动发送邮件。发送前你可以查看、编辑或删除草稿。

草稿可能包含简短的诊断摘要，以便支持人员理解上下文：App 版本、macOS 版本、处理器架构、Auto Blackout 状态、内建镜像状态和当前 App 状态。 只有当你发送邮件时，Xintech LLC 才能看到你的邮件地址。

错误报告草稿还会包含 MacHelper Pro 最近诊断日志的有限摘录。这些日志可能包括本地显示状态、App 状态详情、Wi-Fi 网络名称、接入点地址、 信号强度和 Auto Wi-Fi Switch 决策。功能建议和一般反馈草稿不包含日志。

诊断和日志

MacHelper Pro 会向 Apple 本地统一日志系统写入诊断消息，并保留一小段 App 本地诊断日志。这些日志存储在你的 Mac 本地。 选择“报告问题...”会从 App 本地日志中收集仅限 MacHelper Pro 的有限摘录，并插入邮件草稿。

日志摘录会在可用时使用最近 20 分钟的 MacHelper Pro 日志。如果近期条目太少，App 可能回看最多 24 小时，并包含最近的 MacHelper Pro 条目，最多 180 行和 16,000 个字符。除非你选择发送，否则不会发送任何内容，并且你可以先删除日志摘录。

你的选择

  • 你可以不发送反馈草稿，也可以编辑草稿或删除诊断详情。
  • 发送错误报告前，你可以删除诊断摘要或日志摘录。
  • 你可以在 Sparkle 更新偏好设置中关闭自动更新检查。
  • 你可以从 MacHelper Pro 菜单开启或关闭 Auto Wi-Fi Switch。
  • 你可以在 macOS 系统设置中授予、拒绝或撤销 Wi-Fi 详情所需的位置权限。
  • 你可以在试用期间不激活许可证而使用 Pro 显示控制。
  • 你可以从许可证窗口停用这台 Mac。
  • 你可以使用 macOS 常规偏好设置管理工具移除本地 App 偏好设置。
