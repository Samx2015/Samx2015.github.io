Політика конфіденційності MacHelper Pro

Дата набрання чинності: 2 червня 2026 року

MacHelper Pro — локальна утиліта рядка меню macOS для керування вбудованим дисплеєм у дзеркальних конфігураціях Mac і, за бажанням, допомоги Mac перемикатися на сильнішу збережену Wi‑Fi мережу.

Перекладені версії надаються для зручності. Англійська версія є авторитетною версією цієї політики.

Немає облікового запису програми Немає MacHelper Pro входу або профілю користувача.
Ніякої аналітики Немає відстеження подій, відстеження сеансів, рекламних пікселів або телеметрії використання.
Немає паролів Wi-Fi Wi-Fi перемикання використовує macOS деталі мережі; MacHelper Pro не читає паролі.

Збір даних

MacHelper Pro не збирає, не зберігає, не продає, не ділиться та не передає особисту інформацію за допомогою функцій керування відображенням. Xintech LLC не отримує інформацію про ваші дисплеї, налаштування, використання чи пристрій із програми, якщо ви навмисно не надішлете її через службу підтримки електронною поштою або не активуєте ліцензію.

  • Обліковий запис MacHelper Pro не потрібен.
  • Немає аналітики, SDK для відстеження, SDK для реклами або SDK для звітів про збої.
  • На Xintech LLC фонова телеметрія не надсилається.

Локальні дані

MacHelper Pro може зберігати невелику кількість локальних даних про налаштування та обмежений локальний журнал діагностики на вашому Mac. Це використовується лише для забезпечення стабільної поведінки програми між запусками та підготовки чернеток звітів про помилки, коли ви вирішуєте повідомити про помилку.

Місцеві переваги можуть включати:

  • Чи ввімкнено або вимкнено автоматичне затемнення.
  • Чи слід увімкнути вбудоване дзеркальне відображення під час запуску.
  • Спеціальні комбінації клавіш для дій меню MacHelper Pro.
  • Чи ввімкнено Auto Wi-Fi Switch та порогове значення сили сигналу.
  • Ідентифікатор локального дисплея, який використовується для запам’ятовування бажаного зовнішнього дисплея для віддзеркалення.
  • Невеликий обмежений журнал діагностичних повідомлень MacHelper Pro, які використовуються для чернеток звітів про помилки.
  • Маркери початку пробної версії та завершення її дії зберігаються локально та в Keychain для керування пробним доступом.
  • Ліцензійний ключ і Lemon Squeezy ідентифікатор примірника ліцензії, які зберігаються в Keychain вашого Mac, якщо ви активуєте ліцензію.
  • Несекретні мітки часу ліцензії, як-от час останньої успішної перевірки, що використовується для пільгової поведінки в автономному режимі.

Додаток використовує API дисплея macOS для перевірки макета дисплея, виявлення дзеркального відображення, налаштування вбудованої яскравості дисплея, тимчасового затемнення активних дисплеїв, коли ви вибираєте "Dim Displays Until Key Press," керування Start at Login, коли його вмикаєте, і відновлення фіксованого безпечного значення яскравості за потреби. Для тимчасового затемнення MacHelper Pro перевіряє, чи була натиснута клавіша лише для відновлення відображення; він не записує назви клавіш, введений текст або історію клавіатури. MacHelper Pro не зберігає вашу попередню яскравість.

Статус Wi-Fi і дозвіл геолокації

Auto Wi-Fi Switch необов’язковий. Якщо ввімкнено, MacHelper Pro використовує загальнодоступні API macOS CoreWLAN для читання поточної назви Wi-Fi, адреси точки доступу, потужності сигналу та діапазону частот, а також для пошуку збережених мереж автоматичного приєднання, які можуть бути кращими альтернативами.

macOS може вимагати дозволу на доступ до місцезнаходження, перш ніж програми зможуть побачити Wi-Fi мережеві назви та адреси точок доступу. MacHelper Pro використовує цей дозвіл лише для Wi-Fi деталей, які використовуються в меню та Auto Wi-Fi Switch. Програма не збирає історію місцезнаходжень, не надсилає деталі Wi-Fi до Xintech LLC і не зчитує, не зберігає та не передає паролі Wi-Fi.

Мережева активність

Звичайні елементи керування дисплеєм MacHelper Pro не потребують підключення до мережі. Програма може звертатися до загальнодоступної стрічки оновлень Sparkle, коли ввімкнено автоматичні перевірки оновлень або коли ви вибираєте "Check for Updates..." у меню.

Перевірки оновлень використовуються лише для того, щоб дізнатися, чи доступна новіша версія. Вони не використовуються для реклами, аналітики, відстеження облікових записів або відстеження використання.

Якщо ви виберете "Activate License...", MacHelper Pro зв’яжеться з Lemon Squeezy, щоб активувати, підтвердити або деактивувати вашу ліцензію. Запити на ліцензію надсилають ваш ліцензійний ключ і назву екземпляра на основі імені локального комп’ютера вашого Mac, наприклад «MacHelper Pro на Mac Сема», щоб ви могли розпізнати та деактивувати цей Mac пізніше.

Lemon Squeezy може повернути метадані ліцензії, продукту, замовлення та клієнта, необхідні для перевірки ліцензії. MacHelper Pro зберігає лише ліцензійний ключ, ідентифікатор екземпляра локальної ліцензії, маркери локальної пробної версії та позначки часу локальної перевірки. Запити на ліцензію не використовуються для аналітики, реклами чи відстеження використання.

Звіти про помилки, запити на функції та відгуки

Пункти меню "Report a Bug...", "Suggest a Feature..." та "Send Feedback..." відкривають чернетку у вашій програмі електронної пошти за умовчанням, адресовану Xintech LLC. Програма не надсилає електронні листи автоматично. Ви можете переглянути, відредагувати або видалити чернетку перед її надсиланням.

Чернетка може містити короткий огляд діагностики, щоб служба підтримки могла зрозуміти контекст: версія програми, версія macOS, архітектура процесора, стан Auto Blackout, стан вбудованого дзеркального відображення та поточний статус програми. Ваша адреса електронної пошти доступна для Xintech LLC, лише якщо ви надсилаєте електронний лист.

Чернетки звітів про помилки також включають обмежений уривок останніх журналів діагностики MacHelper Pro. Ці журнали можуть містити стан локального відображення, деталі статусу програми, Wi-Fi назви мереж, адреси точок доступу, силу сигналу та Auto Wi-Fi Switch рішення. Чернетки запитів на функції та загальні відгуки не містять журналів.

Діагностика та журнали

MacHelper Pro записує діагностичні повідомлення до локальної уніфікованої системи журналювання Apple і зберігає невеликий обмежений локальний журнал діагностики програми. Ці журнали зберігаються локально на вашому Mac. Вибір "Report a Bug..." збирає обмежений MacHelper Pro-тільки витяг із локального журналу програми та вставляє його в чернетку електронної пошти.

Уривок журналу використовує останні 20 хвилин журналів MacHelper Pro, якщо вони доступні. Якщо останніх записів замало, програма може переглянути до 24 годин і включити найновіші записи MacHelper Pro, що не перевищують 180 рядків і 16 000 символів. Нічого не надсилається, доки ви не виберете Надіслати, і ви можете спочатку видалити витяг журналу.

Ваш вибір

  • Ви можете залишати чернетки відгуків ненадісланими, редагувати їх або видаляти діагностичні деталі.
  • Ви можете видалити підсумки діагностики або витяги з журналу перед надсиланням звітів про помилки.
  • Ви можете вимкнути автоматичні перевірки оновлень у параметрах оновлення Sparkle.
  • Ви можете ввімкнути або вимкнути Auto Wi-Fi Switch з меню MacHelper Pro.
  • Ви можете надати, заборонити або скасувати дозвіл на визначення місцезнаходження для деталей Wi-Fi у macOS Налаштуваннях системи.
  • Ви можете використовувати елементи керування екраном Pro протягом пробної версії без активації ліцензії.
  • Ви можете деактивувати цей Mac у вікні ліцензії.
  • Ви можете видалити локальні параметри програми за допомогою звичайних macOS інструментів керування параметрами.
