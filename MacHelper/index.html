A quiet menu-bar app for MacBook display control, keyboard backlight, and Finder context menu actions.

What is it?

Everything it does.

  • Auto Blackout Keeps the built-in MacBook display dark while it mirrors an external display.
  • Display controls Switch mirroring, dim displays until key press, sleep the Mac, and adjust display brightness or contrast from the menu bar.
  • Keyboard backlight Adjust the built-in MacBook keyboard brightness with a slider.
  • Finder context menus Create a new file in the current folder, copy paths, or open Finder selections in your configured editor, terminal, Codex, VSCode, or another compatible app.
  • Privacy basics No account, no analytics, and no Dock window.
What's good about it?

Less fuss around displays and files.

  • Only blacks out when it should dark when mirrored, normal when the MacBook screen is part of your workspace
  • Fast everyday actions switch mirroring, dim until key press, sleep, adjust display levels, create new files, copy paths, and open Finder selections from context menus
  • Local-first by default settings stay on your Mac, feedback opens as a reviewable email draft, and there is no analytics SDK
