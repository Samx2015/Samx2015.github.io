MacHelper Pro 隱私權政策

生效日期：2026 年 6 月 2 日

MacHelper Pro 是一款本機優先的 macOS 選單列工具，用於在 Mac 鏡像設定中控制內建顯示器， 並可選擇協助 Mac 切換到訊號更強的已儲存 Wi-Fi。

本譯文僅為方便閱讀而提供。英文版本是本政策的權威版本。

無需 App 帳號 無需 MacHelper Pro 登入或使用者個人資料。
無分析追蹤 不進行事件追蹤、工作階段追蹤、廣告像素或使用遙測。
不讀取 Wi-Fi 密碼 Wi-Fi 切換使用 macOS 網路詳細資料；MacHelper Pro 不讀取密碼。

資料收集

MacHelper Pro 不會透過顯示管理功能收集、儲存、出售、分享或傳輸個人資訊。除非你主動透過支援郵件傳送資訊或啟用授權， 否則 Xintech LLC 不會從 App 收到關於你的顯示器、設定、使用情況或裝置的資訊。

  • 無需 MacHelper Pro 帳號。
  • 不包含分析、追蹤、廣告或當機回報 SDK。
  • 不會向 Xintech LLC 傳送背景遙測。

本機資料

MacHelper Pro 可能會在你的 Mac 上儲存少量本機偏好設定和有限的本機診斷日誌。這些資料僅用於讓 App 在多次啟動之間保持一致行為， 並在你選擇回報問題時準備錯誤回報草稿。

本機偏好設定可能包括：

  • 自動黑屏是否啟用。
  • 啟動時是否啟用內建顯示器鏡像。
  • MacHelper Pro 選單動作的自訂鍵盤快捷鍵。
  • Auto Wi-Fi Switch 是否啟用及其訊號強度閾值。
  • 用於記住鏡像首選外接顯示器的本機顯示器識別碼。
  • 用於錯誤回報草稿的一小段 MacHelper Pro 診斷訊息日誌。
  • 用於管理試用權限的試用開始和試用到期標記，儲存在本機和鑰匙圈中。
  • 如果你啟用授權，授權金鑰和 Lemon Squeezy 授權實例識別碼會儲存在 Mac 鑰匙圈中。
  • 非秘密的授權時間戳，例如上次成功驗證時間，用於離線寬限行為。

App 使用 macOS 顯示 API 檢查顯示器配置、偵測鏡像、調整內建顯示器亮度、在你選擇「調暗顯示器直到按鍵」時暫時調暗作用中顯示器、 在你啟用時管理登入時啟動，並在需要時恢復固定的安全亮度值。對於暫時調暗，MacHelper Pro 只檢查是否有按鍵按下，以便恢復顯示器； 它不會記錄按鍵名稱、輸入文字或鍵盤歷史。MacHelper Pro 不會儲存你之前的亮度。

Wi-Fi 狀態和位置權限

Auto Wi-Fi Switch 是可選功能。啟用後，MacHelper Pro 使用公開的 macOS CoreWLAN API 讀取目前 Wi-Fi 名稱、 存取點位址、訊號強度和頻段，並掃描可能作為更好替代的已儲存自動加入網路。

macOS 可能要求位置權限，App 才能查看 Wi-Fi 網路名稱和存取點位址。MacHelper Pro 僅將該權限用於選單和 Auto Wi-Fi Switch 所需的 Wi-Fi 詳細資料。App 不收集位置歷史，不向 Xintech LLC 傳送 Wi-Fi 詳細資料，也不讀取、儲存或傳輸 Wi-Fi 密碼。

網路活動

MacHelper Pro 的一般顯示控制不需要網路連線。當自動更新檢查啟用，或你從選單選擇「檢查更新...」時， App 可能會存取公開的 Sparkle 更新 feed。

更新檢查僅用於了解是否有新版本。它不會用於廣告、分析、帳號追蹤或使用情況追蹤。

如果你選擇「啟用授權...」，MacHelper Pro 會聯絡 Lemon Squeezy 來啟用、驗證或停用授權。授權請求會傳送你的授權金鑰， 以及基於 Mac 本機電腦名稱的實例名稱，例如「MacHelper Pro on Sam's Mac」，以便你之後識別並停用該 Mac。

Lemon Squeezy 可能回傳驗證授權所需的授權、產品、訂單和客戶中繼資料。MacHelper Pro 只儲存授權金鑰、本機授權實例識別碼、 本機試用標記和本機驗證時間戳。授權請求不會用於分析、廣告或使用情況追蹤。

錯誤回報、功能建議和意見回饋

「回報問題...」、「建議功能...」和「傳送回饋...」選單項目會在你的預設郵件 App 中開啟一封寄給 Xintech LLC 的草稿。 App 不會自動傳送郵件。傳送前你可以查看、編輯或刪除草稿。

草稿可能包含簡短診斷摘要，以便支援人員理解背景：App 版本、macOS 版本、處理器架構、Auto Blackout 狀態、內建鏡像狀態和目前 App 狀態。 只有當你傳送郵件時，Xintech LLC 才能看到你的電子郵件地址。

錯誤回報草稿也會包含 MacHelper Pro 最近診斷日誌的有限摘錄。這些日誌可能包含本機顯示狀態、App 狀態詳細資料、Wi-Fi 網路名稱、 存取點位址、訊號強度和 Auto Wi-Fi Switch 決策。功能建議和一般回饋草稿不包含日誌。

診斷和日誌

MacHelper Pro 會向 Apple 本機統一日誌系統寫入診斷訊息，並保留一小段 App 本機診斷日誌。這些日誌儲存在你的 Mac 本機。 選擇「回報問題...」會從 App 本機日誌中收集僅限 MacHelper Pro 的有限摘錄，並插入郵件草稿。

日誌摘錄會在可用時使用最近 20 分鐘的 MacHelper Pro 日誌。如果近期條目太少，App 可能回看最多 24 小時並包含最近的 MacHelper Pro 條目，最多 180 行和 16,000 個字元。除非你選擇傳送，否則不會傳送任何內容，而且你可以先刪除日誌摘錄。

你的選擇

  • 你可以不傳送回饋草稿，也可以編輯草稿或刪除診斷詳細資料。
  • 傳送錯誤回報前，你可以刪除診斷摘要或日誌摘錄。
  • 你可以在 Sparkle 更新偏好設定中關閉自動更新檢查。
  • 你可以從 MacHelper Pro 選單啟用或關閉 Auto Wi-Fi Switch。
  • 你可以在 macOS 系統設定中授予、拒絕或撤銷 Wi-Fi 詳細資料所需的位置權限。
  • 你可以在試用期間不啟用授權而使用 Pro 顯示控制。
  • 你可以從授權視窗停用這台 Mac。
  • 你可以使用 macOS 一般偏好設定管理工具移除本機 App 偏好設定。
